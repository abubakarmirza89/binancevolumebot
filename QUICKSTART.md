# 🚀 Vercel Se Azad - Local Server Setup!

## **5 Minute Setup (Zero Coding)**

### **STEP 1: Download Node.js**
👉 Go to: https://nodejs.org/
- Download LTS version
- Click "Next, Next, Next" to install
- Restart your PC

### **STEP 2: Run the Bot**

Open Command Prompt/Terminal in your bot folder:

```bash
node server.js
```

**You should see:**
```
╔════════════════════════════════════════╗
║   🤖 Binance Volume Bot - Local Mode  ║
╚════════════════════════════════════════╝

✓ Server running on: http://localhost:3000
```

### **STEP 3: Open Browser**
```
http://localhost:3000
```

✅ **Bot is live! No 451 errors, direct Binance access!**
